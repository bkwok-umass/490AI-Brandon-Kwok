AI Class Project

This page summarizes my final project for RESECON 490AI (AI in Economics). It provides a brief overview of the topic and links to the main project materials.

Project Overview

In this project, I investigate how AI tools can be applied within an economic or business context to generate insights and support data-driven decision-making. I focus on a specific use case, explain the relevant economic background, and demonstrate how AI or machine learning methods can be used to analyze real or simulated data.

The report describes the motivation, data, methods, and results, while the slide deck highlights the key takeaways for a broader audience. Together, they form a concise portfolio piece showing both technical skills and economic reasoning.
